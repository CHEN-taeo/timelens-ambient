# TimeLens Ambient — 品味与品牌宪章

> 本文档是 TimeLens 的设计北极星。所有 UI、交互、文案、功能取舍，以本文为准。
> 最后更新：2026-06-16

---

## 1. 品牌定位

**TimeLens 不是：** 时间追踪 App、效率警察、数据仪表盘、AI 监视器。

**TimeLens 是：** 桌面上的**时间镜子** —— 有品位的安静伴侣（Warm Restraint）。

**北极星：**

> 让用户更像自己，而不是更强烈地意识到有个 App 在监视自己。

**一句话：**

安静存在，有意图时才丰富；「被看见」的一瞬和「温柔告别」的一瞬，是唯一需要精心设计的时刻。

**参考气质：** Braun / Dieter Rams、Obsidian Dark、Raycast、Linear、Apple Dynamic Island。

---

## 2. 品味基调（Taste Tone）

| 维度 | 要 | 不要 |
|------|-----|------|
| 视觉 | 磨砂玻璃、1px 微白边、近黑非纯黑 | 白底 SaaS、强 drop shadow、霓虹泛滥 |
| 色彩 | 60-30-10；accent 只给 ● 和顶边线 | 字、边、背景同时上色 |
| 字体 | Inter 弱对比 + JetBrains Mono（tabular-nums） | 到处 bold、数字宽度抖动 |
| 动效 | 200–500ms 弹簧；液滴生长/吸回 | 线性 500ms fade、teleport 弹窗 |
| 文案 | 短、具体、像朋友 | 审判、AI 腔、「你失败了」 |
| 数据 | 可解释、可撤销 | 黑盒 AI、「我注意到你…」 |

**品牌禁忌：**

- 红色 modal、全屏打断
- 专注过程中的确认对话框
- 每秒狂跳数字
- 白底企业 Dashboard
- 无法解释的分类

## 3. 产品签名（Signature Elements）

这些是 TimeLens 独有的识别符号，不可随意替换：

1. **● 即产品** — 最小态以呼吸点为核心；品牌 = 那颗会呼吸的点
2. **液滴展开** — Focus 模式从胶囊向下生长，不是窗口 teleport
3. **距离协议** — 公共 / 社交 / 个人 三态；永不未经邀请闯入
4. **间（Ma）协议** — 停顿是性格，不是 bug
5. **诚实镜像** — 每条分类必须可展示「因为：…」，支持 5s 撤销

---

## 4. 交互协议（Proxemics Protocol）

```
Ambient（默认）      ● 43m                     — 公共区，零打扰
Peek（悬停 400ms）   ● 编程 · 43m              — 社交区，只瞥见，不展开大板
Focus（单击）        液滴展开 Bento             — 个人区，用户明确邀请
收起                 Esc / 点击外部 / 顶栏 ✕

● 单击（未来）       直接开始番茄，零面板
● 双击（未来）       展开 Focus
长按 ●（未来）       收工仪式

拖动                 仅专用拖柄，不与点击冲突
```

**禁止：**

- hover 自动展开完整 Bento 面板
- 整个胶囊区域 `-webkit-app-region: drag`（会与点击冲突）
- 动画过程中频繁 resize Electron 窗口导致鼠标丢失

## 5. 间（Ma）时机规范

| 时刻 | 延迟 / 节奏 |
|------|------------|
| 悬停 → Peek | 400ms 静止后再显示分类 |
| 单击 → Focus 展开 | 200ms 定住 → 液滴向下生长 |
| 内容分层淡入 | 0 / 80 / 160 / 240ms |
| 偏离警告 | 连续偏离 ≥90s 才 amber 呼吸；≥5min 才标记番茄失败 |
| 番茄完成反馈 | 300ms 静 → 轻胀 → 浮出文案 → 1.5s 消失 |
| 日结后 | 停止刷新，● 进入「睡眠」呼吸 |

**弹簧参数（参考）：** `stiffness: 280, damping: 22–28`

---

## 6. 峰终定律（Peak-End）

不必优化每一秒的平均体验。只设计两个记忆点：

### 峰 1 — 被看见

系统准确知道用户在做什么（应用 + 时长 + 意图），一行轻反馈，不是通知。

示例：`TimeLens 设计 · 已 36min`

### 峰 2 — 完整的一轮

番茄完成：300ms 静 → 胶囊轻胀 → `+1 🍅 · 完整 · 25min` → 5s 可撤销。

### 终 — 具体告别

日结（如 22:00），具体、温暖、非泛泛而谈：

> 今天主线 62%，编程 4h12m，比昨天多 42min。晚安。

之后 ● 缓慢呼吸，停止刷新。

## 8. 文案规范

**语气：** 镜子，不是法官。

| 场景 | 要 | 不要 |
|------|-----|------|
| 偏离 | `偏离 2min` | `你在摸鱼！` |
| 中断 | `番茄已中断 · 撤销 5s` | `确定要放弃吗？` |
| 日结 | `比昨天多 42min` | `今日效率评分 72` |
| 分类 | `因为：bilibili.com · 规则 #7` | `AI 判定为娱乐` |

**日结 / 主线（未来）：**

- 每日一条「主线意图」，不是 Todo 列表
- 显示「主线推进 %」，不是「做了 47 件事」

## 9. 功能哲学

### 三层栈

```
Layer 1 — 感知（ActivityWatch）     真实行为，不评判
Layer 2 — 意图（主线 + 番茄 + 日历）  计划 vs 现实
Layer 3 — 镜子（可解释 + 峰终 + Ma）  让用户更像自己
```

### MVP 必做

- ● 最小态 + Peek + Click Focus
- ActivityWatch 实时分类
- 番茄 + 偏离检测（环境警告，无 modal）
- 可解释分类 + 撤销
- 液滴动效 + 预分配窗口
- **Dead reckoning ●** — 颜色编码一日进度（见 `TIMELENS-SCIENCE.md`）
- **OODA Orient** — Focus 面板预填番茄意图

### 阶段 2+

- Google Calendar 轻碰撞（amber 呼吸，非罚单）
- 日结告别 + ● 睡眠
- 个人规则学习（纠错永久记住）
- 「时间天气」隐喻（上午晴 / 下午有云）

### 不做

- 登录 / 云同步（MVP）
- 复杂 Todo / Kanban
- 黑盒 AI 分类无解释
- 白底报表 Dashboard
- 专注中 modal 确认

## 10. 竞品差异

| 竞品 | TimeLens |
|------|----------|
| RescueTime / Real Soon | 打开看报表 → 看 ● 就够 |
| AI 自动分类 | 规则可解释 + 用户可纠错 |
| 效率警察 | 时间镜子 |
| Dashboard | 环境胶囊 |
| 优化平均体验 | 只设计峰与终 |

---

## 11. 所有者品味关键词（Owner DNA）

- **克制** — Less, but better
- **诚实** — 数据可解释，不 manipulation
- **Ambient** — 在屏幕上，不像 App
- **有温度** — 具体告别，非冷冰冰统计
- **像物件** — 胶囊是桌面上的东西，不是网页
- **灵动岛哲学** — 生命感、液滴、呼吸，不是弹窗

## 12. 实现检查清单

新功能 / PR 合并前，自问：

- [ ] 默认态是否足够安静（≤2 个信息块）？
- [ ] 丰富态是否需用户明确意图（click）？
- [ ] 动效是否有「间」和 spatial continuity？
- [ ] 是否避免 modal / 确认框（改用 undo）？
- [ ] 分类是否可解释？
- [ ] 视觉是否遵循 60-30-10 和 glass token？
- [ ] 文案是否像镜子，不像法官？
- [ ] 是否设计了峰或终，而不是又加一个平均态功能？
- [ ] ● 颜色是否遵循 dead reckoning（非仅分类色）？
